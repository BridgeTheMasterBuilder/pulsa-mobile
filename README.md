## Getting Started

To get a local copy up and running follow these steps:

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/BridgeTheMasterBuilder/pulsa-mobile.git
   ```
2. Open project in Android Studio

3. Build and run the project
